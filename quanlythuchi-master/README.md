## Clone

```
git clone https://github.com/hvtheer/quanlythuchi.git
```

## Install

- XAMPP: tự xem trên mạng, rồi cài composer
```
composer install
```
- Cái này chạy ở 1 terminal
```
cp .env.example .env
php artisan migrate
```
```
php artisan serve
```
- Cái này chạy ở 1 terminal song song không là không chạy được đâu
```
npm install
```
```
npm run dev
```
